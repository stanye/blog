php抽象类:

  1. PHP5支持抽象类和抽象方法。
  2. 抽象类不能被实例化，必须先继承再实例化。

  3. 继承抽象类, 子类必须全部实现抽象类中的方法, 而且public、private、protected只能高不能低

    以上例程会输出：
    ConcreteClass1
    FOO_ConcreteClass1
    ConcreteClass2
    FOO_ConcreteClass2
    抽象类的子类中的普通方法执行方式和其他类相同
